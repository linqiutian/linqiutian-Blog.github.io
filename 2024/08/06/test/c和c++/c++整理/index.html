  • cout - 标准输出
  • cerr - 标准错误输出
  • clog - 标准日志输出

二、内存分区模型

在C++中内存主要分为：

  • 代码区
  • 全局区
  • 栈区
  • 堆区

1、概念

四、new/delete 和 malloc/free

1、new 申请空间

2、delete 释放空间

3、区别（面试题）

  • malloc/free 是C语言库函数，new/delete 是C++运算符
  • new 会调用构造函数，delete 会调用析构函数
  • new 返回具体类型指针，malloc 返回 void*
